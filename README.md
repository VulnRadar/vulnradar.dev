# VulnRadar

Professional web vulnerability scanner. Completely free, forever.

**100+ security checks** for SQL injection, XSS, CSRF, authentication issues, and more. Scan any public URL in seconds.

## Quick Links

- **[Documentation](/docs)** - API reference, setup guide, and guides
- **[GitHub](https://github.com/RejectModders/VulnRadar)** - Source code and contribution guidelines
- **[Report a Bug](https://github.com/RejectModders/VulnRadar/issues)** - Help us improve

## Features

- 100+ automated security checks with minimal false positives
- Real-time scanning with detailed reports
- Export to PDF or JSON
- Scheduled scans and webhooks
- Two-factor authentication
- Team collaboration with role-based access
- REST API with API key authentication
- Dark mode by default

## Getting Started

1. **Sign up** at [vulnradar.app](/signup) - it's free
2. **Scan a URL** to see real-time vulnerability detection
3. **View the docs** at [/docs](/docs) for API integration and advanced features

## Tech Stack

Built with Next.js 15, React 19, TypeScript, Tailwind CSS, PostgreSQL, and more.
