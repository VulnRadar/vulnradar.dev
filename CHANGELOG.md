# Changelog

## [v2.2.0] - 2026-03-31

### Backend Improvements
- Enhanced database performance and optimization
- Improved error handling and logging across API endpoints
- Optimized data processing pipelines

### API Enhancements
- Refined request validation and sanitization
- Improved API response consistency and reliability
- Enhanced webhook handling and delivery mechanisms

### UI/UX Improvements
- Updated component styling and visual consistency
- Improved user interface responsiveness
- Enhanced accessibility features

### Security
- Patched multiple SSRF (Server-Side Request Forgery) vulnerabilities
- Fixed insufficient password hashing implementation
- Resolved ReDoS (Regular Expression Denial of Service) vulnerabilities
- Improved input validation and output sanitization
- Enhanced certificate validation and TLS security
- Removed insecure random number generation

### Dependencies
- Updated lucide-react to v6 for improved security
- Added react-icons for expanded icon library support
- Updated bcryptjs for secure password hashing

For more information about these changes and security updates, please refer to the project documentation.
